Ferramentas Utilizadas

(voltar)

Para começar o meu aprendizado em HTML foi ensinado que antes, eu precisaria de algumas ferramentas.

Uma delas era um navegador. Eu já estava utilizando o Google Chrome para fazer a aula, então seria ele mesmo.

A outra era uma ferramenta que me permitiria escrever o HTML. Muitas opções existem, porém nesse curso optamos pelo Visual Studio Code já que ele é open source.

Tags

(voltar)

Já dentro do VS code eu aprendi a estrutura básica de HTML. É necessário um html, um head e um body. Na head teremos as informações pré-carregadas e no body temos o que será visivel no navegador.

Aprendi também que as tags precisam ser abertas e fechada, salvo algumas exceções como a de input e img.

Além disso, fui ensinado sobre os atributos que as tags podem possuir. Posso utilizar type, style, id, class, src, width, height e muitos outros.

Textos

(voltar)

Há também algumas tags que me permitiriam alterar o texto. Utilizei headers como h1 e h2, aprendi como deixar em italico com i e negrito com strong.

Aprendi também que para referenciar ao que alguém disse, posso usar blockquote para centralizar a mensagem.

Se eu precisar colocar um asterisco ou alguma letra de trademark, eu posso utilizar sup para atingir meu objetivo

Listas Ordenadas e Não Odernadas

(voltar)

Há dois tipos de lsitas que eu posso utilizar, as ordenadas e as não ordenadas.

Para as ordenadas, a tag utilizada é a ol (ordered list). Ela vai enumerar os meus itens para mim.

Para as não ordenadas eu utilizo ul (unordered list). Nesse caso, os meus itens serão indicados por um símbolo que eu poderei personalizar com CSS.

Em ambos os casos, os meus itens serão descritos linhas a linha com a tag li (list item).

Links

(voltar)

Para inserir um link em html utilizaremos da tag a. Ela possui o atributo href no qual eu insiro o link de minha escolha.

A tag a serve como uma âncora que liga o termo colocado dentro dela e o link para outro site.

Eu consigo determinar se ao clicar o usuário vai ser mandado para uma nova aba ou se ele vai utilizar da aba atual.

Para isso, basta utilizar o atributo target e colocá-lo como _self caso deseja que o link abra nessa aba ou _blank caso queira que abra em outra.

Mais um detalhe interessante é o de fazer aparecer um balão de texto ao passar o mouse por cima de um link, com o atributo title.
